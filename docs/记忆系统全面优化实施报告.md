### P1-1: 记忆滑窗分块 ⭐⭐⭐⭐

**文件**: [memory/mod.rs Line 154-201](file:///F:/rust/Ox/crates/ox-core/src/memory/mod.rs#L154-L201)

**实施内容**:
```rust
fn split_with_overlap(text: &str, max_chunk_len: usize, overlap_ratio: f32) -> Vec<String> {
    if text.len() <= max_chunk_len {
        return vec![text.to_string()];
    }
    
    let mut chunks = Vec::new();
    let step = (max_chunk_len as f32 * (1.0 - overlap_ratio)) as usize;
    let mut start = 0;
    
    while start < text.len() {
        let end = (start + max_chunk_len).min(text.len());
        
        // Try to break at word boundary
        let chunk_end = if end < text.len() {
            text[start..end].rfind(' ').map(|pos| start + pos).unwrap_or(end)
        } else {
            end
        };
        
        chunks.push(text[start..chunk_end].to_string());
        start = chunk_end + step - (max_chunk_len as f32 * overlap_ratio) as usize;
    }
    
    chunks
}
```

**集成到 format_memory_context**:
```rust

pub fn update_with_llm_feedback_batch(
    &self,
    feedbacks: Vec<(String, f32)>,  // (memory_id, score)
    project_id: Option<&str>,
) {
    // Collect all updated nodes
    let mut updated_nodes = Vec::new();
    
    for (memory_id, score) in &feedbacks {
        if let Ok(mut node) = self.fetch_memory_by_id(memory_id, store) {
            // Update scores, depth, etc.
            // ...
            updated_nodes.push(node);
        }
    }
    
    // Use insert_batch for efficient bulk update
    if !updated_nodes.is_empty() {
        store.insert_batch(&updated_nodes)?;  // Single transaction
    }
}
```

### 1. 智能分层截断

**创新点**: 首次将 LLM 评分用于记忆压缩决策

```
评分 ≥ 8.0 → 400 chars（高质量，保留完整）
评分 6.0-7.9 → 280 chars（中等质量，标准长度）
评分 < 6.0 → 200 chars（低质量，精简）
未评分 → 280 chars（默认长度）
```

**优势**:
- 自适应不同质量的记忆
- 平衡 token 效率和语义完整性
- 无需人工配置阈值
